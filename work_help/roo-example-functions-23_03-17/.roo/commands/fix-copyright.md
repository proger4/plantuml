Добавь или проверь copyright в PHP-файле.

Аргумент: {{args}}

Инструкции:
- Не редактируй файл напрямую.
- Всегда вызывай скрипт:

python3 .roo/scripts/fix_copyright.py "{{args}}"

- После выполнения:
  - если файл изменён — сообщи путь;
  - если уже корректен — сообщи это;
  - если не найден — сообщи ошибку.
